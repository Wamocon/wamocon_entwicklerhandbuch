Schnellzugriff

🚀 Neue App starten

Kapitel 08 – Entwicklungsprozess → Template Repo klonen → Anforderungsdokument in Copilot laden

🗄️ Supabase einrichten

Kapitel 10 – Supabase → Per E-Mail beantragen → env.local befüllen → Migrationen mit Copilot

🌐 Deployen

Kapitel 11 – Vercel → Vercel Project ID als Secret → Push auf main → automatisches Deploy

🤖 Copilot nutzen

Kapitel 13 – KI & Copilot → @planner → @developer → @reviewer → PR erstellen
