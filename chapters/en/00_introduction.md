# Chapter 00 – Introduction

## What is the WAMOCON Developer Handbook?

The WAMOCON Developer Handbook is the central reference for all developers who create and
operate apps, landing pages, or infrastructure projects within the WAMOCON organisation.
It describes the standards, processes, and tools recommended for development.

## Target Audience

This handbook is intended for:

- Developers building WAMOCON apps using GitHub Copilot
- Administrators managing Supabase or Vercel projects
- All stakeholders applying WAMOCON processes (Wave Rhythm, Jira, Requirements Document)

## Development Philosophy

All app development at WAMOCON is carried out entirely with **GitHub Copilot**.
Copilot is the primary development tool, from planning through implementation to
migrations and deployment configurations.

## Binding Nature

All standards described in this handbook represent the recommended approach for
WAMOCON apps. Deviations are possible and can be documented in the respective
project repository.
